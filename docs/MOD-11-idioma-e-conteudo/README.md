# MOD 11 — Idioma e conteúdo

**Status:** Planejado  
**User Stories:** 4  
**Produto:** Fodinha / Oh Hell Game V2  
**Fonte:** [PRD mestre](../../PRD-SEPARACAO-WEB-MOBILE.md)

## Objetivo

manter PT/EN completos e independentes de layout.

## Diretriz técnica do módulo

Traduções devem ter paridade PT/EN, sem strings de produto hardcoded ou chaves criadas apenas para controlar layout.
