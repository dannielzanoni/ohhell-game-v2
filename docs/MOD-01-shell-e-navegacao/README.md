# MOD 01 — Shell e navegação

**Status:** Planejado  
**User Stories:** 5  
**Produto:** Fodinha / Oh Hell Game V2  
**Fonte:** [PRD mestre](../../PRD-SEPARACAO-WEB-MOBILE.md)

## Objetivo

fornecer navegação própria por plataforma com as mesmas rotas e capacidades.

## Diretriz técnica do módulo

Usar CSS mobile-first, breakpoint único em 768 px, navegação acessível e preservação de estado durante resize.

## Dependências

MOD 00 — Fundação de plataforma e contratos.
